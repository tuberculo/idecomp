Arquivos (rv0, jan, …)

Os nomes dos arquivos a serem utilizados pelo DECOMP são armazenados na classe:

class Arquivos(data=<cfinterface.data.sectiondata.SectionData object>)[código fonte]

Armazena os dados de entrada do NEWAVE referentes ao arquivo arquivos.dat.

Esta classe lida com informações de entrada do NEWAVE e que deve se referir aos nomes dos demais arquivos de entrada utilizados para o caso em questão.

property arquivos: List[str]

Os nomes dos arquivos utilizados.

Retorna:

Os arquivos na mesma ordem em que são declarados

Tipo de retorno:

List[str]

property caminho: str

Caminho para os executáveis do DECOMP.

property dadger: str

Nome do arquivo de dados gerais utilizado pelo DECOMP.

property dadgnl: str

Nome do arquivo com os dados das usinas térmicas GNL.

property hidr: str

Nome do arquivo de cadastro dos dados das hidrelétricas.

property mlt: str

Nome do arquivo com as médias mensais de longo termo (MLT).

property perdas: str

Nome do arquivo com as perdas no sistema.

property vazoes: str

Nome do arquivo de vazões incrementais afluentes.
