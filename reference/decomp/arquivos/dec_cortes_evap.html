Cortes da Evaporação Linear (dec_cortes_evap.csv)

Os cortes para aproximação da evaporação linear são armazenados na classe:

class DecCortesEvap(data=<cfinterface.data.blockdata.BlockData object>)[código fonte]

Arquivo com os cortes da evaporação linear do DECOMP.

property tabela: DataFrame | None

A tabela de dados que está contida no arquivo.

  • periodo (int)

  • indice_usina (int)

  • nome_usina (str)

  • submercado (int)

  • ree (int)

  • derivada_cota_area (float)

  • derivada_volume_cota (float)

  • volume_referencia_hm3 (float)

  • evaporacao_referencia_hm3 (float)

  • coeficiente_volume (float)

  • rhs_volume (float)

Retorna:

A tabela como um dataframe

Tipo de retorno:

pd.DataFrame | None
