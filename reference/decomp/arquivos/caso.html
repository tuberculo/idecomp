Caso (caso.dat)

A informação do nome do arquivo que define o caso é armazenada na classe:

class Caso(data=Ellipsis)[código fonte]

Armazena os dados de entrada do NEWAVE referentes ao nome do arquivo que define os arquivos do caso.

Esta classe lida com informações de entrada fornecidas ao NEWAVE e que podem ser modificadas através do arquivo caso.dat.

property arquivos: str | None

Caminho para o arquivo arquivos.dat de entrada do DECOMP.

Retorna:

O caminho para o arquivo

Tipo de retorno:

str | None
