Interface de Programação para o DECOMP

Versão: 0.0.55

Data: 01/03/2023

O idecomp é um pacote Python para manipulação dos arquivos de entrada e saída do programa DECOMP, desenvolvido pelo CEPEL e utilizado para os estudos de planejamento e operação do Sistema Interligado Nacional (SIN).

O idecomp oferece:

  • Meios para leitura dos arquivos de entrada e saída do DECOMP

  • Armazenamento e processamento de dados otimizados com o uso de NumPy

  • Facilidades para estudo e análise dos dados utilizando DataFrames de Pandas

  • Dados estruturados em modelos com o uso do paradigma de orientação a objetos (OOP)

  • Utilidades de escritas dos arquivos de entrada do DECOMP para elaboração automatizada de estudos
